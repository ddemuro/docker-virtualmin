# docker-virtualmin
This is the dockerfile and other files to run Virtualmin in docker.

## Included packages
  1. Virtualmin
  2. Curl
  3. NFS
  4. Postfix configuration
  5. LDAP

We use LDAP mapping to ensure the same users in all platform.

## Used for:
This dockerfile is used for takelan deployments.

## Volumes
  1. Logs
  2. Homes
